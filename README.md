# KbSpas Web App
I built this app for KbSpas, a retired GTA-based lash and brow business. I was new to web development at the time so it's a bit shit. Check it out at: [kbspas.vercel.app](kbspas.vercel.app)

**To run locally:**
1. Run `cd client`, `npm install`, `npm run dev` in terminal to start the frontend
2. Create a .env file in the root and client folder using their respective example files to fill in environment variables
3. Open a second terminal and run `npm install`, `npm run dev` to start the backend
